4. loginup.php 파일 생성

login.php 파일을 만들어 해당 코드는 동일하게 입력하지만 $_SESSION[“username”] 세션 안에 대숙이라는 값을 대입 한다.



입력한 뒤 html 창에 localhost:8000/login.php 를 입력한 뒤 다시 주소창에 localhost:8000/loginup.php 입력해 가면 아무것도 뜨지 않지만 다시 뒤로 가기 버튼을 누루게 되면 해당 사진처럼 바뀐 것을 알 수 있다. 이것은 세션을 통해 대입된 값이 저장된 상태에서 다시 세션을 이용하기에 전에 저장된 값도 같이 출력되는 것이다. 즉, up 파일은 저장만 하는 것이고, 그냥 login 파일은 저장된 값을 읽는 것이다.
